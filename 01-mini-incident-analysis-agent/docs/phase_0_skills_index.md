# Phase 0 Skills Index

This index lists the local agent skills available for the Mini Incident Analysis Agent project.

- **project-guardrails**: Überprüft Scope und Claims. Nutzen bei Architektur-Diskussionen. Nicht für Code-Erzeugung.
- **scope-fixing**: Hilft bei Scope- und Entscheidungsdokumenten. Nutzen bei Scope-Anpassungen. Nicht für Feature-Creep.
- **repo-setup-planning**: Planung für spätere Verzeichnisstruktur. Nutzen beim initialen Setup. Nicht für Runtime-Logik.
- **documentation-planning**: Plant README und Projekt-Docs. Nutzen bei Doku-Entwürfen. Schreibt keinen Code.
- **data-context-planning**: Plant Runbooks und Sample Incidents. Nutzen für Datenaufbereitung. Nicht für echte DB-Anbindungen.
- **schema-output-planning**: Plant Output-Felder. Nutzen für die Report-Struktur. Keine komplexen JSON-Hierarchien.
- **tool-layer-mcp-lite**: Plant die kleine MCP-nahe Tool-Layer. Nutzen bei Tool-Konzeption. Nicht für full-scale MCP Plattformen.
- **incident-agent-planning**: Plant späteren Agent-Workflow. Nutzen bei der Ablauflogik. Keine Multi-Agenten-Architektur.
- **e2e-validation-planning**: Plant Demo- und Testpfade. Nutzen zur Validierung der DoD. Keine fetten CI/CD Setups.
- **interview-demo-readiness**: Sichert ehrliche Darstellung. Nutzen bei README und Intro-Präsi. Keine übertriebenen Claims.
- **scope-creep-check**: Warnt vor unnötigen Erweiterungen. Nutzen wenn neue Ideen aufkommen. Verhindert Scope-Ausweitungen.
